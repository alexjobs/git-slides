What is GIT?

It is a tool that manages and tracks different versions of software source code or other text content.

It is a version control system (VCS), a source code manager (SCM) or a revision control system (RCS)

Why GIT?

  • Speed
  • Strong support for non-linear development (thousands of parallel branches)
  • Fully distributed
  • Able to handle large projects like the Linux kernel efficiently (speed and data size)
