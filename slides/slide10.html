describe

if you want to have a human-readable name to go with a commit, you can run git describe on that commit. Git gives you the name of the nearest tag with the number of commits on top of that tag and a partial SHA–1 value of the commit you're describing


  $ git describe master
  v1.6.2-rc1-20-g8c5b85c
