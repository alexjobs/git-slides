Patching

git-format-patch

git-format-patch - Prepare patches for e-mail submission


  # Extract commits between revisions R1 and R2
  $ git format-patch -k --stdout R1..R2
  # Extract all commits which are in the current branch but not in the origin branch
  $ git format-patch origin
  # Extract all commits that lead to origin since the inception of the project
  $ git format-patch --root origin
  # Extract three topmost commits from the current branch
  $ git format-patch -3

apply

git-apply - Apply a patch to files and/or to the index


  $ git apply --whitespace=warn "patch"
  $ git apply --whitespace=fix "patch"

am

git-am - Apply a series of patches from a mailbox


  $ git am *.patch
  $ git am 0002-allow-users-to-be-locked.patch

Always use am
