Role: Project Manager
Leads the planning of the project, coordinates interactions with the stakeholders, and keeps the project team focused on meeting the project objectives.
Role Sets: Roles
Relationships
Additionally Performs:
  • Analyze Architectural Requirements
  • Define Vision

  • Develop the Architecture
Modifies:
  • Iteration Plan
  • Project Plan
  • Risk List
  • Status Assessment
  • Work Items List
Main Description

This role:

  • Applies management knowledge, skills, tools and techniques to a broad range of tasks in order to deliver the desired result for a particular project in a timely fashion.
  • Is accountable for the outcome of the project and the acceptance of the product by the customer.
  • Responsible for the evaluation of project's risks, and controling these risks through mitigation strategies. 
Staffing
Skills

A person performing this role needs the following skills:

  • Good in presentation, facilitation, communication, and negotiation.
  • Leadership and team building capabilities.
  • Thorough experience in the software development lifecycle  to teach, guide and support it.
  • Proficient in conflict resolution and problem solving techniques.
