OpenUP/Basic is organized into four major areas of content, Communication and Collaboration, Intent, Solution, and Management, also known as sub-processes.



 

OpenUP/Basic is an iterative software development process with the following characteristics:

  • Minimal: Only fundamental process content is included
  • Complete: It can be manifested as an entire process to build a system
  • Extensible: It can be used as a foundation to add or tailor process content
