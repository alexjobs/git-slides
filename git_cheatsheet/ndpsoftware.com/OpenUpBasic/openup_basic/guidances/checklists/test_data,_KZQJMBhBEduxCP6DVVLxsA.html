Checklist: Test Data
This checklist provides questions to verify that test data is created in a consistent and complete manner.
Relationships
Related Elements
  • Test Script
Check Items
Has test data been provided for the test cases to be implemented
Does the test data support boundary testing
Has the system under test been tested under conditions with sufficient data, insufficient data, and the wrong type of data
Has test data been organized into production and generated data
Has test data been updated to apply to the appropriate iteration
Is the test data complete for the given test effort
Has data been identified for input numeric variables, output numeric variables, arrays, and vectors
